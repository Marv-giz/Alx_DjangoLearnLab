# LibraryProject

## Description
LibraryProject is a basic Django project created to demonstrate the setup and structure of a Django application.

## Requirements
- Python
- Django

## Setup Instructions
1. Install Django:
   pip install django

2. Run the development server:
   python manage.py runserver

3. Open your browser and go to:
   http://127.0.0.1:8000/

## Project Structure
- manage.py: Django command-line utility
- settings.py: Project configuration
- urls.py: URL routing
